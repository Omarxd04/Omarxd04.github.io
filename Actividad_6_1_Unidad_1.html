Siglas

OSI (Open System Interconnection)

Este se trata de un modelo conceptual que describe como se comunican los sistemas de RED y como se envian datos.

IP (Internet Protocol)

En conclusion las direcciones IP es el conjunto de una serie de numeros asignados a cada dispositivo conectado a una red informatica o a internet.

TCP(Transmission Control Protocol)

Este es un protocolo de red importante ya que permite que dos anfritiones (HOSTS) se conecten e intercambien flujos de datos.

UDP (User Datagram Protocol)

Es un protocolo de comunicacion el cual se utiliza en Internet para trasmisiones sujetas a limitaciones temporales.

SSH (Secure Shell)

El cual es un protocolo de administración remota que le permite al usuario controlar y modificar sus servidores remotos a través de Internet.

FTP(File Transfer Protocol)

Este se utiliza para transferir todo tipo de archivos entre equipos conectados a una red.

SMTP (Simple Mail Transfer Protocol)

Es un protocolo simple de transferencia de correo

POP (Post Office Protocol)

Es un protocolo que sirve para sincronizar el correo de Gmail con cuialquier cliente de correo compatible.

DHCP (Dynamic Host Configuration Protocol)

El cual es un conjunto de reglas para dar direcciones IP y opciones de configuraciones a ordenadores y estaciones de trabajo en una red.

DNS (Domain Name System)

Este sistema es básicamente la agenda telefónica de la web.

HTTP (Hypertext Transfer Protocol)

Es un protocolo el cual nos permite realizar una peticion de datos y recursos, como pueden ser archivos HTML.

ARP (Address Resolution Protocol )

Convierte dinamicamente las direcciones de internet en direcciones hardware.

MAC (Medium Access Control)

Es la dirección física y propia de cada dispositivo.

LAN (Local Area Network)

Esta se trata de una red de comunicación que conecta dispositivos en una área geográfica cercano.
