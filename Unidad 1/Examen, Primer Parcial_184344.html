Ecuación cuadrática


Una ecuación de segundo grado o ecuación cuadrática de una variable es aquella que tiene de expresión general:

Ecuación de segundo grado

ax2+ bx + c = 0, a ≠ 0

Donde x es la variable; mientras que a, b y c son valores constantes y corresponde a los coeficientes:

    a.cuadrático (a)
    b.lineal (b)
    c.independiente (c)

Para una ecuación cuadrática con coeficientes reales o complejos existen siempre dos soluciones (x1 y x2), no necesariamente distintas, llamadas raices que puede ser reales o complejas.Fórmula general para la ontención de raices: i = (-b±√(b2-4ac)÷(2a)).Se utiliza el simbolo ± para indicar las dos soluciones

#include

int main () {

std:: cout << "Hola mundo";
return 0;

}

Si no has programado antes, este programa debe sonar casi tan ilegible
